# Personal Finance Dashboard

A full-stack web application for tracking income and expenses, viewing spending
by category, and charting balance over time.

## Tech Stack

- **Backend:** Spring Boot 3.x / Java 17
- **Frontend:** React (Vite) with Tailwind CSS and Recharts
- **Database:** H2 (development), MySQL (production)

## Status

Phase 2a — Backend under construction (entity and persistence layer).
